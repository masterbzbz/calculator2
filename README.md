# 簡易計算機 (Simple Calculator)

這是一個使用原生 HTML、CSS 和 JavaScript 構建的網頁版簡易計算機。介面設計簡潔現代，支援基本的數學運算功能。

## ✨ 功能特色

- **基本運算**：支援加 (`+`)、減 (`-`)、乘 (`×`)、除 (`/`) 四則運算。
- **編輯功能**：
  - `C` (Clear)：清除所有輸入，重置計算機。
  - `DEL` (Delete)：刪除最後一個輸入的字元。
- **小數點支援**：可進行小數運算。
- **錯誤處理**：當發生運算錯誤（如格式不正確）時，會顯示 "Error" 提示。
- **響應式設計**：按鈕佈局直觀，使用 Grid 排版，適應不同螢幕大小。

## 🛠️ 使用技術

- **HTML5**：建構網頁結構。
- **CSS3**：設計樣式與佈局 (Flexbox & Grid)。
- **JavaScript**：處理運算邏輯與 DOM 操作。

## 🚀 如何使用

1. 下載此專案的所有檔案 (`index.html`, `style.css`, `script.js`)。
2. 雙擊 `index.html` 檔案，或使用瀏覽器開啟它。
3. 您現在可以在瀏覽器中直接使用計算機了！
